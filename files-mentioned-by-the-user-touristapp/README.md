# TripSmart Travel Buddy

A produced smart travel buddy app built from the provided frontend idea.

## What is included

- Polished responsive travel dashboard frontend.
- Node backend using only built-in modules.
- SQLite database stored at `data/tripsmart.sqlite`.
- Persistent users, saved destinations, generated trips, hotel bookings, and reviews.
- Seeded destinations, places, hotels, nearby services, and demo account.

## Run

```powershell
npm start
```

Open:

```text
http://localhost:4173
```
